## 概要

このプロジェクトは、ボードゲーム「Ito」と互換性のあるゲームをWeb上で遊べるようにしたものです。

## モチベーション

友人たちとItoをオンライン上で遊びたかったが、既存の公開されているWebアプリには問題があった。
友人たちとItoを楽しむために自作する。

## 主要機能

- カード配布
- お題定期
- お題AI生成
- チャット
- 結果発表

## 遊び方

1. ルームマスターがルームを作成
2. 招待URLか参加コードを使用してルームに参加
3. ゲームスタート

## こだわり

先発でItoのようなゲームを公開しているサイトが複数存在しました。  
後発の本プロジェクトでは私が感じた以下の不満点を解決しています。  
特にお題の自動提供とItoのルールに忠実に従いました。

- お題は自分で探す必要がある
- チャット機能がない
- 独自ルールが追加されいている
- UIがシンプルすぎる
